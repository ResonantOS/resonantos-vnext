# Legacy Planning Documents

This directory contains historical planning documents, task specifications, and migration plans preserved for contextual reference. These documents describe planning intent at specific points in time and may describe directions that were never implemented or were later superseded.

## Why These Are Preserved

- **Planning trail** — record of what was considered and when.
- **Task archaeology** — understanding the evolution of feature planning.
- **Migration context** — historical migration and restructuring plans.

## Current Contents

- `ADDON-MIGRATION-PLAN.md` — addon migration planning.
- `BUILD-PLAN.md` — build system planning.
- `TASK.md` — task planning document.
- `TASK-P1-vector-embeddings.md` through `TASK-P5-sdk-documentation.md` — phase-based task specifications.
- `slides/` — presentation artifacts (3-slide deck: HTML + 3 PNG images).

## Usage

- Consult these when understanding the planning history of specific features.
- Do not treat these as current roadmap — see `docs/FEATURE_BACKLOG.md` and `docs/PROJECT_STATUS.md` for current direction.
- Task documents may describe work that was abandoned or superseded.

# Legacy Reports

This directory contains historical reports preserved for contextual reference. These documents are not actively maintained and may describe states of the codebase that no longer exist.

## Why These Are Preserved

- **Context for future investigations** — understanding why certain decisions were made.
- **QA and audit trail** — evidence of testing and review that occurred.
- **Red-team and security analysis** — historical security posture documentation.
- **Browser integration history** — record of Camofox and Chromium browser work.

## Current Contents

- `BROWSER-FIX-REPORT.md` — historical browser fix documentation.
- `CAMOFOX-CORRECTION-PLAN.md` — Camofox correction planning.
- `CAMOFOX-EDGE-CASE-REPORT.md` — Camofox edge case analysis.
- `CAMOFOX-INTEGRATION-REPORT.md` — Camofox integration documentation.
- `CAMOFOX-REQUIREMENTS-GAP-ANALYSIS.md` — Camofox requirements gap analysis.
- `CODEBASE-AUDIT-REPORT.md` — historical codebase audit.
- `RED-TEAM-REPORT.md` — red-team security assessment.
- `RESONATOR-FIX-REPORT.md` — Resonator fix documentation.

## Usage

- Consult these when investigating historical decisions related to browser integration, security posture, or QA processes.
- Do not treat these as current-state documentation.
- If information from these documents is needed in active docs, extract and verify it before incorporating.
